Loki is a log aggregation system inspired by Prometheus. 
It is easy to operate, as it does not index the content of the Kubernetes logs but sets labels for log streams.
The metadata you use (object labels) can be used in Loki for scraping kubernetes logs.
If you use Grafana for metrics, Loki will allow you to have a single point of management for both logging and monitoring.
